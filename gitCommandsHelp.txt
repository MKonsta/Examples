инициализировать проект в текущей папке  git init

-----------Настройки git-а---------
имя пользователя git config --global user.name "username"
и маил git config --global user.email "user@mail.com" "username"
------------------------------------------------------

статус git status

добавление отслеживаемого файла git add pom.xml

---------------Добавление файлов в игнор------------------------------
1. Создаем в корне нашего проекта файл .gitignore
В этом файле мы указываем, какие файлы мы не хотим использовать (отслеживать)
в нашем проекте
.idea/*
*.class

*.war
*.jar
*.ear
-------------------------------------------------------

Добавление всех файлов в отслеживаемые git add --all

коммит проекта git commit -m "commit's name"

Далее нужно создать новый репозиторий в аккаунте, скопировать url
и прописать в консоли git remote add example https://github.com/MKonsta/Examples
тут 'example' - это будет имя репозитория

посмотреть репозиторий  (имя) в текущей папке git remote


Залить все изменения на удаленный репозиторий git push имя_репозитория master

------------------Ветки-------------------------

Создание версии git tag "version_1.0"

создание новой ветки git branch personClassBranch

посмотреть какие ветки сейчас есть git branch (активная в настоящий момент ветка помечена звездочкой)

переключиться на другую ветку - git checkout personClassBranch. С этой ветки такжк можно
коммитить, и это не будет отражаться на основной ветке master

слиться с веткой, находясь в ветке мастер git merge personClassBranch

удалить ветку git branch -D personClassBranch

посмотреть историю изменений git log
